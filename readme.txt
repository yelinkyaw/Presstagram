== Description ==

Presstagram is a plugin that will integrate the Instagram post to Wordpress. The plugin will automatically check from linked Instagram account and new Wordpress posts will be created according to the Instagram posts. The Presstagram can allow integrating Instagram Hashtags to Wordpress Tags. And several Wordpress post configuration can be changed from the plugin's setting.

== Installation ==

1. Extract the plugin archive
1. Upload plugin files to your /wp-content/plugins/ directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configuration can be changed from Presstagram menu from Settings
